Installation Guide

1

Download the DMG File

Click the download button above to get the DMG file.

2

Open the DMG File

Locate the downloaded file in your Downloads folder and double-click to open it.

3

Drag to Applications

Drag the smileybrooms icon to the Applications folder shortcut in the DMG window.

4

First Launch

The first time you open the app, right-click on the app icon and select "Open" to bypass macOS security warnings about apps from unidentified developers.

System Requirements

  • macOS 10.14 (Mojave) or later
  • 4GB RAM minimum (8GB recommended)
  • 200MB free disk space
  • Internet connection required for booking services
