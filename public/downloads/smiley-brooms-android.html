Installation Instructions:

1
Download the APK

Click the button above to download the APK file to your device.

2
Enable Unknown Sources

Go to Settings > Security > Unknown Sources and enable it to allow installation of apps from sources other than the Play Store.

3
Install the App

Open the downloaded APK file and tap "Install" when prompted.

4
Open the App

Once installation is complete, tap "Open" to launch the app.

Note: For Android 8.0 and above, you may need to enable installation permissions for the specific browser you're using to download the APK.
