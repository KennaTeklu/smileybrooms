smileybrooms for Linux

Download our Linux application to book cleaning services, track your appointments, and manage your cleaning schedule from your Linux machine.

Installation Instructions

Installing on Ubuntu/Debian-based distributions

After downloading the .deb package, you can install it using either the graphical package manager or the command line:

Using the command line:

sudo dpkg -i ~/Downloads/smileybrooms.deb
sudo apt-get install -f # To resolve any dependencies

Using the graphical interface:

  1. Navigate to your Downloads folder
  2. Double-click on the smileybrooms.deb file
  3. The Software Center should open
  4. Click "Install" and enter your password when prompted

Installing on Fedora/RHEL-based distributions

After downloading the .rpm package, you can install it using either the graphical package manager or the command line:

Using the command line:

sudo dnf install ~/Downloads/smileybrooms.rpm

Using the graphical interface:

  1. Navigate to your Downloads folder
  2. Double-click on the smileybrooms.rpm file
  3. The Software Center should open
  4. Click "Install" and enter your password when prompted

Using the AppImage (works on most Linux distributions)

AppImage is a universal package format that runs on most Linux distributions without installation:

  1. Download the AppImage file
  2. Make it executable with the following command:
chmod +x ~/Downloads/smileybrooms.AppImage
  1. Run the AppImage by double-clicking it or using the command line:
~/Downloads/smileybrooms.AppImage

System Requirements

  • Any modern Linux distribution (Ubuntu 20.04+, Fedora 34+, etc.)
  • 4GB RAM minimum (8GB recommended)
  • 200MB free disk space
  • X11 or Wayland display server
  • Internet connection required for booking services
