Installation Guide

1

Download the Installer

Click the download button above to get the installer file.

2

Run the Installer

Locate the downloaded file in your Downloads folder and double-click to run it.

3

Security Warning

If Windows SmartScreen appears, click "More info" and then "Run anyway" to proceed with the installation.

4

Follow Installation Wizard

Follow the on-screen instructions to complete the installation process.

5

Launch the App

Once installation is complete, launch smileybrooms from your Start menu or desktop shortcut.

System Requirements

  • Windows 10 or Windows 11
  • 4GB RAM minimum (8GB recommended)
  • 200MB free disk space
  • .NET Framework 4.7.2 or later
  • Internet connection required for booking services
